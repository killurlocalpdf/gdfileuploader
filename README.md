# gdfileuploader
I made a File Uploader, that turns any data into Geometry Dash level, which you can download and turn back into real data!
I got inspired from SweepSweep2's video (https://www.youtube.com/watch?v=oENqzFJ3TgI), and I tried to re-create it, sooo..

Tutorial and other stuff in the files instructions.txt and gd_file_uploader.py.
